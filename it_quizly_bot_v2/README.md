# 🧠 IT Quizly — Telegram Bot

Викторина и казино для айтишников. Отвечай на технические вопросы, зарабатывай SVOллары и спускай их в играх против других игроков.

## Возможности

### Викторина
- 6 тем: алгоритмы, SQL, сети, DevOps, frontend, угадай язык
- Таймер на каждый вопрос
- Стрик и таблица лидеров
- Пояснения к каждому ответу
- Ежедневный бонус +100 💵

### Игры (PvP на SVOллары)

| Игра | Механика |
|------|----------|
| 🎲 Кости | Telegram dice — честный бросок, больше — победил |
| 💣 Минное поле | Открывай клетки, забирай выигрыш пока не взорвался |
| 🔫 Русская рулетка | По очереди нажимаешь курок, вероятность растёт |
| 🎯 Снайпер | Прячешь число, соперник угадывает за 6 попыток |
| 🎯 Дартс | 3 броска Telegram dice, сумма очков |
| 🎩 Напёрстки | 5 раундов, угадай под каким напёрстком шарик |
| 🏰 Подземелье | 5 этажей, монстры и сокровища, больше золота — победил |
| 🚀 Космос | 5 раундов угроз, правильный манёвр — меньше урона |
| 🕵️ Детектив | Мафия vs детектив, 3 алиби, 3 вопроса, обвиняй |

### Магазин
- 🌈 Радужный ник в таблице лидеров
- ⏱ Бонус к таймеру на вопросы

## Установка

### 1. Создай бота
Напиши [@BotFather](https://t.me/BotFather), отправь `/newbot`, скопируй токен.

### 2. Настройка
```bash
npm install
cp .env.example .env
# Заполни BOT_TOKEN и ADMIN_ID в .env
```

### 3. Запуск
```bash
npm start          # обычный запуск
npm run dev        # с авто-перезапуском (nodemon)
```

## Переменные окружения

| Переменная | Описание |
|-----------|----------|
| `BOT_TOKEN` | Токен от BotFather |
| `ADMIN_ID` | Telegram ID администратора |
| `DB_PATH` | Путь к базе данных (по умолчанию `./quiz.db`) |

## Структура проекта

```
├── bot.js           # Вся логика бота и обработчики
├── db.js            # SQLite: схема, функции для всех игр
├── questions.js     # Банк IT-вопросов
├── web_questions.js # Банк Web-вопросов
├── logger.js        # Логирование событий
├── package.json
├── .env.example
└── quiz.db          # База данных (создаётся автоматически)
```

## Добавление вопросов

В `questions.js` или `web_questions.js`:

```js
{
  category: "algorithms", // algorithms | interview | databases | devops | guess_lang | web
  label: "🧮 Алгоритмы",
  question: "Текст вопроса?",
  options: ["Вариант A", "Вариант B", "Вариант C", "Вариант D"],
  answer: 0, // индекс правильного ответа (0–3)
  explanation: "Пояснение к правильному ответу.",
}
```

## Деплой на Railway

1. Залей код в GitHub
2. Создай новый проект на [Railway](https://railway.app) из репозитория
3. Добавь Volume → Mount Path: `/app/data`
4. Добавь переменные:
   - `BOT_TOKEN` = токен бота
   - `ADMIN_ID` = твой Telegram ID
   - `DB_PATH` = `/app/data/quiz.db`
5. Задеплой — бот работает 24/7

> При каждом деплое бот уведомляет всех пользователей с просьбой написать `/start`.
