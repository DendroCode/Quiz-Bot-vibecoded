# 🤖 IT-Quiz Telegram Bot

Викторина для айтишников с темами: алгоритмы, SQL, DevOps, Frontend и другие.

## Возможности

- 6 тем + случайные вопросы
- 30+ вопросов в банке
- Система очков и таблица лидеров
- Профиль игрока с точностью и стриком
- Пояснения к каждому ответу
- SQLite база данных (без внешнего сервера)

## Установка

### 1. Создай бота в Telegram

1. Напиши [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot` и следуй инструкциям
3. Скопируй токен

### 2. Настрой проект

```bash
# Установи зависимости
npm install

# Создай .env файл
cp .env.example .env

# Вставь токен в .env
# BOT_TOKEN=ваш_токен_здесь
```

### 3. Запуск

```bash
# Обычный запуск
npm start

# Режим разработки (с авто-перезапуском)
npm run dev
```

## Команды бота

| Команда    | Описание              |
|------------|-----------------------|
| `/start`   | Главное меню          |
| `/play`    | Начать викторину      |
| `/top`     | Таблица лидеров       |
| `/profile` | Твой профиль          |

## Структура проекта

```
├── bot.js          # Основная логика бота
├── db.js           # Работа с SQLite
├── questions.js    # Банк вопросов
├── package.json
├── .env            # Токен (не коммитить!)
└── quiz.db         # База данных (создаётся автоматически)
```

## Добавление вопросов

Открой `questions.js` и добавь объект в массив:

```js
{
  category: "algorithms", // algorithms | interview | databases | devops | guess_lang | web
  label: "🧮 Алгоритмы",
  question: "Текст вопроса?",
  options: ["Вариант A", "Вариант B", "Вариант C", "Вариант D"],
  answer: 0, // Индекс правильного ответа (0-3)
  explanation: "Пояснение к правильному ответу.",
}
```
